## Google Sheets : le tableur collaboratif

<AffiliateLink href="https://sheets.google.com/">Google Sheets</AffiliateLink> est le tableur en ligne le plus utilisé. Il fonctionne dans le navigateur, sur tous les systèmes, sans rien installer. Un compte Google suffit.

La collaboration en temps réel est son point fort. Plusieurs personnes modifient le même fichier en même temps. Commentaires, historique des versions, partage par lien : tout est inclus. Pour un usage en équipe, c'est le choix le plus simple.

Le tableur gère les formules classiques, les graphiques, les tableaux croisés dynamiques et les filtres. Google ajoute aussi des fonctions propres comme GOOGLEFINANCE ou IMPORTDATA pour récupérer des données en direct depuis le web.

<Callout type="success" title="Les points forts">
Gratuit avec 15 Go de stockage. Collaboration en temps réel. Accessible partout. Plus de 400 fonctions. Extensions via Google Apps Script. Import/export .xlsx.
</Callout>

<Callout type="warning" title="Le bémol">
Moins performant qu'Excel sur les gros fichiers (au-delà de 10 000 lignes, ça rame). Pas de macros VBA. Le mode hors ligne via Chrome est limité. Vos données sont chez Google.
</Callout>

## LibreOffice Calc : le tableur libre et complet

<CloudinaryImage src="blog-screenshots/libreoffice-writer-home" alt="LibreOffice, suite bureautique gratuite avec le tableur Calc" />

<AffiliateLink href="https://fr.libreoffice.org/">LibreOffice Calc</AffiliateLink> est le tableur de la suite LibreOffice. Open source, gratuit, disponible sur Windows, Mac et Linux. Pas de compte à créer, pas de connexion internet.

Calc est le tableur gratuit le plus proche d'Excel en termes de fonctionnalités. Il gère les macros (en Basic, pas VBA), les tableaux croisés dynamiques, les graphiques avancés et les formules matricielles. Le format natif est ODF, mais l'import/export .xlsx fonctionne.

Pour un usage bureautique classique (budgets, comptabilité, listes de données), Calc remplace Excel sans problème. Les différences apparaissent sur les macros VBA complexes et les fichiers très lourds.

<Callout type="success" title="Les points forts">
Gratuit et open source. Fonctionne hors ligne. Compatible Windows, Mac, Linux. Gère les tableaux croisés dynamiques, les macros et les formules avancées. Communauté active.
</Callout>

<Callout type="warning" title="Le bémol">
L'interface paraît datée. Pas de collaboration en temps réel. La compatibilité VBA est partielle. Lent au démarrage sur Mac. Pas d'app mobile native.
</Callout>

## Zoho Sheet : le tableur cloud sous-estimé

<CloudinaryImage src="blog-screenshots/zoho-writer-home" alt="Zoho, suite bureautique en ligne avec tableur gratuit" />

<AffiliateLink href="https://www.zoho.com/fr/sheet/">Zoho Sheet</AffiliateLink> fait partie de la suite Zoho Workplace. Le tableur est gratuit pour un usage personnel (jusqu'à 5 Go de stockage). Il fonctionne dans le navigateur et sur mobile.

Zoho Sheet propose des fonctionnalités qu'on ne trouve pas ailleurs gratuitement. Les automatisations (macros en JavaScript), l'assistant IA Zia, les connexions à des sources de données externes. Pour un tableur cloud gratuit, c'est celui qui offre le plus de fonctions avancées.

La collaboration fonctionne bien : édition en temps réel, commentaires, verrouillage de cellules, historique. L'interface est propre et bien organisée.

<Callout type="success" title="Les points forts">
Gratuit (5 Go). Automatisations en JavaScript. IA intégrée (Zia). Collaboration en temps réel. Import/export Excel. App mobile. Plus de 350 fonctions.
</Callout>

<Callout type="warning" title="Le bémol">
Moins connu, donc moins de ressources et tutoriels en ligne. L'univers Zoho est vaste mais peut sembler complexe au début. La version gratuite est limitée à 5 utilisateurs par organisation.
</Callout>

## OnlyOffice : la compatibilité .xlsx avant tout

<CloudinaryImage src="blog-screenshots/onlyoffice-home" alt="OnlyOffice, tableur gratuit avec compatibilité Excel" />

<AffiliateLink href="https://www.onlyoffice.com/fr/">OnlyOffice</AffiliateLink> est un éditeur open source qui travaille nativement en format .xlsx. Pas de conversion, pas de perte de mise en forme. Si vous échangez des fichiers Excel avec d'autres personnes, c'est un avantage réel.

Le tableur couvre les besoins courants : formules, graphiques, tableaux croisés dynamiques, filtres, mise en forme conditionnelle. L'interface ressemble à celle d'Excel avec le système de ruban.

La version Desktop Editors est gratuite. Pour la collaboration en ligne, il faut déployer le serveur documentaire (auto-hébergé) ou utiliser OnlyOffice Cloud.

<Callout type="success" title="Les points forts">
Open source. Format natif .xlsx (pas de conversion). Interface moderne style Excel. Auto-hébergeable. Gratuit en version bureau.
</Callout>

<Callout type="warning" title="Le bémol">
Moins de fonctions qu'Excel ou LibreOffice Calc. La collaboration nécessite un serveur. Pas de macros VBA. L'app mobile est basique.
</Callout>

## WPS Spreadsheets : l'interface Excel à l'identique

<CloudinaryImage src="blog-screenshots/wps-office-home" alt="WPS Office, tableur gratuit avec interface identique à Excel" />

<AffiliateLink href="https://www.wps.com/">WPS Spreadsheets</AffiliateLink> reproduit l'interface d'Excel presque pixel par pixel. Les onglets, le ruban, les raccourcis clavier : tout est au même endroit. Si vous connaissez Excel, vous n'aurez rien à réapprendre.

La compatibilité avec les fichiers .xlsx est excellente. Les mises en page, les formules et les graphiques sont bien conservés. WPS gère aussi les macros VBA (dans une certaine mesure).

La version gratuite affiche des publicités. C'est le prix à payer. L'éditeur est chinois (Kingsoft), ce qui pose des questions de confidentialité pour certains utilisateurs.

<Callout type="success" title="Les points forts">
Interface identique à Excel. Excellente compatibilité .xlsx. Disponible sur tous les appareils (Windows, Mac, Linux, Android, iOS). Léger et rapide. Support partiel des macros VBA.
</Callout>

<Callout type="warning" title="Le bémol">
Publicités dans la version gratuite. Éditeur chinois (Kingsoft), questions sur la confidentialité. Pas open source. Certaines fonctions avancées réservées à la version Pro.
</Callout>

## Questions fréquentes

### Excel est-il toujours payant ?

Oui. Excel seul n'est plus vendu séparément. Il faut passer par Microsoft 365 (69 euros par an). La version en ligne (Excel Online) est gratuite mais très limitée : pas de macros, pas de tableaux croisés dynamiques avancés, pas de Power Query.

### Google Sheets peut-il remplacer Excel ?

Pour les usages courants (budgets, listes, formules basiques, graphiques), oui. Pour les fichiers volumineux, les macros VBA, Power Query ou Power Pivot, Excel reste devant. Google Sheets est aussi dépendant d'une connexion internet.

### Quel est le meilleur tableur gratuit pour un usage pro ?

Google Sheets si vous travaillez en équipe. LibreOffice Calc si vous avez besoin de fonctions avancées hors ligne. Le choix dépend de votre façon de travailler : cloud ou local, seul ou en équipe.

### Les tableurs gratuits sont-ils compatibles avec Excel ?

Tous lisent et écrivent le format .xlsx. La compatibilité est bonne pour les fichiers simples. Les problèmes apparaissent avec les macros VBA, les mises en page très spécifiques et les formules Excel exclusives (comme XLOOKUP dans les anciennes versions de Calc).
