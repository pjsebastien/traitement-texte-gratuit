Firefox reste un bon navigateur. Mais entre les changements de direction de Mozilla, la baisse de parts de marché et certaines décisions controversées sur la télémétrie, des utilisateurs cherchent autre chose. Un navigateur open source, rapide, qui respecte la vie privée.

Voici cinq alternatives crédibles.

## Brave : la vie privée par défaut

<CloudinaryImage src="blog-screenshots/brave-home" alt="Brave, navigateur web axé sur la confidentialité" />

<AffiliateLink href="https://brave.com/">Brave</AffiliateLink> bloque les pubs et les traqueurs dès l'installation. Pas besoin d'ajouter uBlock Origin ou une autre extension. Le blocage est natif, et il est agressif.

Le navigateur est basé sur Chromium (le moteur open source de Google Chrome). Les extensions du Chrome Web Store fonctionnent directement. La compatibilité avec les sites web est excellente.

Brave intègre aussi un moteur de recherche maison (<AffiliateLink href="https://search.brave.com/">Brave Search</AffiliateLink>), un VPN payant, et un système de récompenses en cryptomonnaie (les BAT). Ce dernier point divise : certains trouvent ça malin, d'autres trouvent ça inutile.

<Callout type="success" title="Ce qui le rend unique">
Blocage natif des pubs et traqueurs, compatible extensions Chrome, performances solides. Le passage depuis Firefox ou Chrome est simple. L'app mobile est aussi bien protégée que la version bureau.
</Callout>

<Callout type="warning" title="Le bémol">
Le système de récompenses BAT (cryptomonnaie) peut agacer. Brave est basé sur Chromium, ce qui renforce la domination de Google sur le web. Et certains reprochent à Brave d'avoir remplacé les pubs des sites par les siennes.
</Callout>

## Chromium : le navigateur brut

<CloudinaryImage src="blog-screenshots/chrome-home" alt="Chromium, le projet open source derrière Google Chrome" />

<AffiliateLink href="https://www.chromium.org/">Chromium</AffiliateLink> est le projet open source sur lequel Google Chrome est construit. C'est Chrome sans les services Google : pas de synchronisation Google, pas de rapports d'utilisation envoyés automatiquement, pas de mises à jour automatiques.

C'est un choix technique. Chromium s'adresse à ceux qui veulent un navigateur dépouillé, sans surcouche commerciale. La compatibilité web est identique à Chrome.

Le problème principal : les mises à jour. Chromium ne se met pas à jour tout seul. Sur certaines distributions Linux, c'est géré par le gestionnaire de paquets. Sur Windows, il faut télécharger manuellement les nouvelles versions.

<Callout type="success" title="Ce qui le rend unique">
C'est la base de Chrome, sans les mouchards de Google. Totalement open source. Compatible avec les extensions Chrome. Parfait pour les développeurs qui veulent un navigateur propre.
</Callout>

<Callout type="warning" title="Le bémol">
Pas de mises à jour automatiques (sauf sur Linux via le gestionnaire de paquets). Pas de synchronisation native. L'expérience utilisateur est plus brute que Firefox ou Brave. Ce n'est pas fait pour le grand public.
</Callout>

## LibreWolf : Firefox sans la télémétrie

<AffiliateLink href="https://librewolf.net/">LibreWolf</AffiliateLink> est un fork de Firefox. Le code est celui de Firefox, mais nettoyé. La télémétrie est désactivée. Les paramètres de confidentialité sont réglés au maximum dès l'installation. Le moteur de recherche par défaut est DuckDuckGo.

uBlock Origin est préinstallé. Les fonctionnalités "phoning home" (les connexions que Firefox fait vers les serveurs Mozilla) sont supprimées. LibreWolf ne contacte personne au démarrage.

Si vous aimez Firefox mais détestez devoir modifier 30 paramètres dans about:config pour protéger votre vie privée, LibreWolf fait ce travail pour vous.

<Callout type="success" title="Ce qui le rend unique">
Firefox sans télémétrie, sans DRM activé par défaut, avec uBlock Origin préinstallé. Les extensions Firefox fonctionnent. La transition depuis Firefox est transparente. Parfait pour ceux qui veulent un Firefox "propre" simplement.
</Callout>

<Callout type="warning" title="Le bémol">
Les mises à jour arrivent avec un léger décalage par rapport à Firefox (quelques jours). Pas de synchronisation Firefox Sync (désactivée volontairement). Sur Windows, l'installation passe par un installeur tiers ou Winget. La communauté est plus petite que celle de Firefox.
</Callout>

## Tor Browser : l'anonymat total

<AffiliateLink href="https://www.torproject.org/">Tor Browser</AffiliateLink> est basé sur Firefox ESR (la version longue durée). Mais il route tout le trafic à travers le réseau Tor, ce qui masque votre adresse IP et rend le suivi quasi impossible.

Ce n'est pas un navigateur du quotidien. La navigation est lente (le trafic passe par plusieurs relais). Certains sites bloquent les connexions Tor. Et les CAPTCHAs deviennent votre quotidien.

Mais pour des cas précis (journalisme, recherche sensible, accès à des contenus censurés), Tor Browser est le seul outil vraiment fiable.

<Callout type="success" title="Ce qui le rend unique">
Anonymat réel grâce au réseau Tor. Résistance au fingerprinting (tous les utilisateurs ont la même empreinte). Accès au réseau .onion. C'est l'outil de référence pour la navigation anonyme.
</Callout>

<Callout type="warning" title="Le bémol">
Lent. Beaucoup de sites affichent des CAPTCHAs ou refusent les connexions Tor. Pas adapté au streaming ou aux téléchargements. L'usage quotidien est pénible. Il complète un navigateur principal, il ne le remplace pas.
</Callout>

## Vivaldi : la personnalisation poussée

<AffiliateLink href="https://vivaldi.com/">Vivaldi</AffiliateLink> est créé par l'ancien fondateur d'Opera. Le navigateur est basé sur Chromium, mais l'interface est entièrement personnalisable : position des onglets, panneaux latéraux, raccourcis clavier, thèmes, chaînes de commandes.

Vivaldi intègre un client mail, un calendrier, un lecteur de flux RSS et un traducteur. Tout est dans le navigateur. Pas besoin d'extensions pour ces fonctions.

Le code de l'interface est propriétaire, mais le moteur (Chromium) est open source. Vivaldi se positionne entre l'open source pur et le navigateur grand public.

<Callout type="success" title="Ce qui le rend unique">
Personnalisation extrême de l'interface. Client mail, calendrier et RSS intégrés. Gestion avancée des onglets (empilage, mosaïque). Compatible extensions Chrome. Bloqueur de pubs natif.
</Callout>

<Callout type="warning" title="Le bémol">
L'interface n'est pas open source (seul le moteur Chromium l'est). La richesse des options peut perdre les utilisateurs qui veulent juste un navigateur simple. La consommation mémoire est élevée avec beaucoup d'onglets ouverts.
</Callout>

## Questions fréquentes

### Firefox est-il encore un bon navigateur ?

Oui. Firefox reste rapide, respectueux de la vie privée (comparé à Chrome) et activement maintenu. Les raisons de chercher une alternative sont souvent liées à la télémétrie activée par défaut, aux partenariats commerciaux de Mozilla, ou à la volonté d'avoir un navigateur basé sur Chromium pour la compatibilité.

### LibreWolf ou Firefox, lequel choisir ?

LibreWolf est Firefox avec les paramètres de confidentialité réglés au maximum. Si vous prenez le temps de configurer Firefox manuellement (about:config, extensions), le résultat est similaire. LibreWolf fait gagner du temps, mais vous perdez Firefox Sync.

### Brave est-il vraiment open source ?

Le code source de Brave est disponible sur GitHub. Vous pouvez le compiler vous-même. Le navigateur est bien open source. Le système de récompenses BAT est optionnel et peut être désactivé.

### Tor Browser est-il légal ?

Oui, dans la plupart des pays. Utiliser Tor est légal en France, en Belgique, en Suisse et au Canada. C'est un outil de protection de la vie privée, pas un outil criminel. Ce que vous faites sur Tor est soumis aux mêmes lois que sur n'importe quel autre navigateur.
