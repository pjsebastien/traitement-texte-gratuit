L'iPad est devenu un vrai outil de travail. Avec un clavier (Magic Keyboard ou pas), il remplace un ordinateur portable pour beaucoup d'usages. Encore faut-il avoir un bon traitement de texte. Voici les meilleures options gratuites.

<Callout type="tip" title="Le résumé en 10 secondes">
**Pages** est déjà installé sur votre iPad et fait le job. **Google Docs** est le choix si vous collaborez avec d'autres personnes. **Zoho Writer** offre une IA gratuite. **WPS Office** imite l'interface Word. **OnlyOffice** est open source et compatible Word.
</Callout>

## Pages : le natif Apple

<AffiliateLink href="https://www.apple.com/fr/pages/">Pages</AffiliateLink> est préinstallé sur tous les iPad. Pas besoin de télécharger quoi que ce soit. L'app est optimisée pour le tactile et prend en charge l'Apple Pencil pour annoter directement dans le document.

Les modèles sont parmi les plus élégants du marché. La synchronisation via iCloud permet de retrouver ses documents sur Mac et iPhone. L'export en .docx, PDF et EPUB est natif.

Pages gère aussi la collaboration en temps réel. Vous partagez un lien, vos collaborateurs éditent depuis leur navigateur ou leur propre appareil Apple.

<Callout type="success" title="Ce qui le rend unique">
Gratuit, préinstallé, optimisé pour l'iPad et l'Apple Pencil. Les modèles sont beaux. La synchronisation iCloud fonctionne sans y penser. L'export en .docx permet d'envoyer des fichiers lisibles par les utilisateurs Word.
</Callout>

<Callout type="warning" title="Le bémol">
La compatibilité Word n'est pas parfaite. Les documents avec des mises en page complexes (tableaux imbriqués, macros, styles avancés) peuvent se déformer à l'import. L'interface "Apple" déroute ceux qui viennent de Word. Pas d'option pour les macros ou le publipostage.
</Callout>

## Google Docs : la collaboration depuis l'iPad

<CloudinaryImage src="blog-screenshots/google-docs-home" alt="Google Docs sur iPad, traitement de texte collaboratif gratuit" />

L'app <AffiliateLink href="https://docs.google.com/">Google Docs</AffiliateLink> pour iPad est gratuite sur l'App Store. Elle fonctionne aussi bien que la version navigateur, avec l'avantage du mode hors ligne (les documents marqués "disponibles hors connexion" restent accessibles sans internet).

La collaboration en temps réel est le point fort. Plusieurs personnes écrivent dans le même document, les commentaires et suggestions fonctionnent, l'historique des versions est complet.

L'app s'intègre avec Google Drive (15 Go gratuits), Gmail et les autres services Google. Si vous utilisez déjà un compte Google, c'est le choix le plus naturel.

<Callout type="success" title="Ce qui le rend unique">
Collaboration en temps réel, 15 Go de stockage, mode hors ligne, intégration Google Drive. L'app iPad est complète et réactive. Les documents Google ne comptent pas dans le quota de stockage.
</Callout>

<Callout type="warning" title="Le bémol">
L'app iPad est moins complète que la version web (certaines fonctions manquent). Le mode hors ligne demande de l'anticiper (il faut marquer les documents à l'avance). La mise en page est basique comparée à Pages ou Word. Les données passent par Google.
</Callout>

## Zoho Writer : l'IA gratuite sur iPad

<CloudinaryImage src="blog-screenshots/zoho-writer-home" alt="Zoho Writer, traitement de texte avec IA gratuite sur iPad" />

<AffiliateLink href="https://www.zoho.com/fr/writer/">Zoho Writer</AffiliateLink> propose une app iPad complète avec une IA (Zia) gratuite. Reformulation, correction grammaticale en français, résumé, traduction en 55+ langues. Tout ça sans payer.

L'app fonctionne hors ligne. Les modifications se synchronisent automatiquement quand la connexion revient. L'interface est propre et adaptée au tactile.

<Callout type="success" title="Ce qui le rend unique">
IA gratuite (Zia) pour la correction, reformulation et traduction. Vérificateur de plagiat intégré. Mode hors ligne natif. Zéro pub, respect du RGPD. C'est le seul traitement de texte gratuit sur iPad qui offre tout ça.
</Callout>

<Callout type="warning" title="Le bémol">
5 Go de stockage gratuit (contre 15 Go chez Google). Moins connu que Google Docs ou Pages, donc moins de tutoriels. L'app iPad est correcte mais moins réactive que Pages sur les très gros documents.
</Callout>

## WPS Office : l'interface Word sur iPad

<CloudinaryImage src="blog-screenshots/wps-office-home" alt="WPS Office, suite bureautique gratuite pour iPad" />

<AffiliateLink href="https://www.wps.com/">WPS Office</AffiliateLink> reproduit l'interface de Microsoft Word quasi à l'identique sur iPad. Si vous venez de Word et que vous cherchez des repères familiers, c'est l'app qu'il vous faut.

La compatibilité avec les fichiers .docx est excellente. Les documents complexes gardent leur mise en page. La version gratuite inclut un traitement de texte, un tableur et un outil de présentation.

<Callout type="success" title="Ce qui le rend unique">
Interface quasi identique à Word. Meilleure compatibilité .docx parmi les apps gratuites. Suite complète (texte + tableur + présentation). Disponible sur toutes les plateformes.
</Callout>

<Callout type="warning" title="Le bémol">
La version gratuite affiche des publicités. Certaines fonctions avancées (PDF, IA) sont payantes. L'app est développée par Kingsoft (entreprise chinoise), ce qui soulève des questions de confidentialité.
</Callout>

## OnlyOffice : l'open source sur iPad

<CloudinaryImage src="blog-screenshots/onlyoffice-ios-home" alt="OnlyOffice Documents, suite bureautique open source pour iPad" />

<AffiliateLink href="https://www.onlyoffice.com/fr/">OnlyOffice</AffiliateLink> est une suite bureautique open source avec une app iPad gratuite. Le traitement de texte ouvre les .docx avec une compatibilité remarquable (meilleure que LibreOffice sur ce point).

L'app se connecte à vos services cloud (Google Drive, Dropbox, OneDrive, Nextcloud) ou à un serveur OnlyOffice auto-hébergé. La collaboration en ligne fonctionne via le serveur documentaire.

<Callout type="success" title="Ce qui le rend unique">
Open source, compatibilité Word excellente, connexion multi-cloud. L'interface est moderne et adaptée au tactile. Pas de pub dans la version gratuite. Le format natif est le .docx (pas de conversion nécessaire).
</Callout>

<Callout type="warning" title="Le bémol">
Moins de fonctionnalités que Pages ou Google Docs sur iPad. Le mode hors ligne est limité (les documents doivent être téléchargés à l'avance). La collaboration nécessite un serveur OnlyOffice (pas aussi simple que Google Docs).
</Callout>

## Questions fréquentes

### Word est-il gratuit sur iPad ?

L'app Microsoft Word pour iPad est gratuite pour la lecture. L'édition nécessite un abonnement Microsoft 365 (69 euros par an). Pour les écrans de moins de 10,1 pouces, l'édition de base est gratuite, mais les iPad récents dépassent tous cette taille.

### Pages peut-il ouvrir des fichiers Word sur iPad ?

Oui. Pages ouvre les .doc et .docx et exporte dans ces formats. La compatibilité est bonne pour les documents simples. Les mises en page complexes peuvent se déformer.

### Quel traitement de texte avec Apple Pencil ?

Pages est le mieux intégré avec l'Apple Pencil. Vous pouvez annoter, dessiner et écrire à la main directement dans le document. Google Docs et WPS Office supportent aussi le Pencil mais de manière plus limitée.

### Peut-on travailler hors connexion sur iPad ?

Oui. Pages et WPS Office fonctionnent nativement hors ligne. Google Docs nécessite de marquer les documents "hors connexion" à l'avance. Zoho Writer gère le hors ligne via son app.
