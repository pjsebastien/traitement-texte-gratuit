AWS domine le marché du cloud. Plus de 30 % de parts de marché mondiales, loin devant tout le monde. Mais dominer ne veut pas dire convenir à tous. Factures surprises, complexité de la console, dépendance à un géant américain : les raisons de chercher ailleurs sont nombreuses.

<Callout type="tip" title="Le résumé en 10 secondes">
**Google Cloud** pour le machine learning et le Big Data. **Microsoft Azure** pour les entreprises déjà sous Windows/Office. **Hetzner** pour le meilleur rapport qualité-prix en Europe. **OVHcloud** pour la souveraineté française. **DigitalOcean** pour la simplicité. **Scaleway** pour le cloud européen nouvelle génération.
</Callout>

## Pourquoi chercher une alternative à AWS ?

AWS propose plus de 200 services. C'est un avantage pour les grandes entreprises. Pour les autres, c'est souvent trop.

- **Tarification opaque** : les factures AWS sont notoirement difficiles à prévoir. Bande passante, requêtes API, stockage, transferts entre régions. Chaque ligne coûte quelque chose
- **Complexité** : la console AWS est un labyrinthe. Configurer un simple serveur web demande de naviguer entre EC2, VPC, Security Groups, IAM, Route 53
- **Dépendance** : plus vous utilisez de services propriétaires AWS (Lambda, DynamoDB, SQS), plus la migration devient difficile
- **Souveraineté des données** : AWS est une entreprise américaine soumise au Cloud Act. Pour les données européennes sensibles, c'est un sujet
- **Prix** : pour un VPS basique, AWS est souvent deux à trois fois plus cher que les alternatives européennes

## Google Cloud Platform : la force de l'IA et du Big Data

<CloudinaryImage src="blog-screenshots/google-cloud-home" alt="Page d'accueil de Google Cloud Platform" />

<AffiliateLink href="https://cloud.google.com/">Google Cloud Platform</AffiliateLink> est le deuxième challenger du marché, derrière Azure. Son point fort : tout ce qui touche à la donnée. BigQuery, Vertex AI, TensorFlow. Si votre activité repose sur l'analyse de données ou le machine learning, Google Cloud est difficile à battre.

Le réseau de Google est aussi un atout majeur. Les temps de latence entre régions sont parmi les plus bas du marché. Le CDN est celui qui sert YouTube et Gmail.

L'offre gratuite est généreuse : 300 dollars de crédit pendant 90 jours, plus un Free Tier permanent sur plusieurs services (f1-micro, Cloud Storage, BigQuery).

<Callout type="success" title="Ce qui le rend unique">
BigQuery pour l'analyse de données massives. Vertex AI et les modèles Gemini intégrés. Réseau mondial parmi les plus rapides. Kubernetes (GKE) est le meilleur du marché, Google étant le créateur de Kubernetes. 300 dollars de crédit gratuit pour démarrer.
</Callout>

<Callout type="warning" title="Le bémol">
La tarification est aussi complexe que celle d'AWS. Le support gratuit est limité (forums communautaires). Moins d'entreprises formées sur GCP que sur AWS. Même problématique Cloud Act que pour AWS.
</Callout>

## Microsoft Azure : le choix des entreprises Windows

<AffiliateLink href="https://azure.microsoft.com/fr-fr/">Microsoft Azure</AffiliateLink> est le premier concurrent d'AWS en parts de marché. Son avantage principal : l'intégration native avec l'univers Microsoft. Active Directory, Office 365, SQL Server, .NET. Si votre entreprise tourne déjà sur des technologies Microsoft, Azure simplifie tout.

Azure propose aussi des régions en France (Paris, Marseille). C'est un point important pour la conformité RGPD, même si la maison mère reste américaine.

Le programme Azure for Students offre 100 dollars de crédit sans carte bancaire.

<Callout type="success" title="Ce qui le rend unique">
Intégration parfaite avec Active Directory, Office 365 et les outils Microsoft. Régions françaises disponibles. Offre hybride (Azure Arc) pour connecter vos serveurs on-premise au cloud. Le plus grand réseau de partenaires et de certifications en Europe.
</Callout>

<Callout type="warning" title="Le bémol">
Tarification aussi complexe qu'AWS. L'interface du portail Azure est lourde et lente. Certains services sont moins matures que leurs équivalents AWS. La documentation est parfois confuse, avec des noms de produits qui changent régulièrement.
</Callout>

## Hetzner : le rapport qualité-prix imbattable

<AffiliateLink href="https://www.hetzner.com/">Hetzner</AffiliateLink> est un hébergeur allemand qui propose des serveurs cloud et dédiés à des prix très agressifs. Un VPS avec 2 vCPU, 4 Go de RAM et 40 Go SSD coûte environ 4,50 euros par mois. Le même type de machine chez AWS coûte trois fois plus.

Les datacenters sont en Allemagne et en Finlande. Hetzner est soumis au RGPD. Pas de Cloud Act.

Le service est volontairement minimaliste. Pas de bases de données managées, pas de fonctions serverless, pas de file d'attente. Vous louez des machines et vous gérez le reste. C'est exactement ce que beaucoup de développeurs recherchent.

<Callout type="success" title="Ce qui le rend unique">
Prix parmi les plus bas du marché (VPS dès 4,50 euros/mois). Serveurs dédiés abordables. 20 To de bande passante incluse. Datacenters en Europe (Allemagne, Finlande). API propre et bien documentée. Pas de frais cachés sur la bande passante.
</Callout>

<Callout type="warning" title="Le bémol">
Pas de services managés (base de données, Kubernetes managé, fonctions serverless). Le support est basique. Moins de régions que les hyperscalers. L'interface d'administration est fonctionnelle mais spartiate. Pas adapté aux entreprises qui veulent du "tout-en-un".
</Callout>

## OVHcloud : la souveraineté française

<AffiliateLink href="https://www.ovhcloud.com/fr/">OVHcloud</AffiliateLink> est le plus grand hébergeur européen. Fondé à Roubaix, l'entreprise possède ses propres datacenters en France, en Allemagne, au Royaume-Uni, au Canada et en Asie.

OVHcloud propose à la fois des VPS, du cloud public (type AWS), du bare metal, du stockage objet, des bases de données managées et même du Kubernetes managé. L'offre est complète.

Un VPS Starter coûte 3,50 euros par mois. Le cloud public propose des instances à partir de 4 euros. Le trafic sortant est inclus (pas de frais de bande passante, contrairement à AWS).

<Callout type="success" title="Ce qui le rend unique">
Entreprise française, datacenters en France. Pas de Cloud Act. Bande passante sortante incluse (c'est énorme face à AWS). Prix compétitifs. Offre complète du VPS au bare metal. Qualifié SecNumCloud pour certaines offres. Active dans le projet Gaia-X.
</Callout>

<Callout type="warning" title="Le bémol">
L'incendie de Strasbourg en 2021 a marqué les esprits (même si les mesures ont été renforcées depuis). L'interface de gestion est en cours de modernisation. Certains services managés sont moins matures que chez AWS ou Google Cloud. Le support peut être lent hors offre premium.
</Callout>

## DigitalOcean : la simplicité avant tout

<AffiliateLink href="https://www.digitalocean.com/">DigitalOcean</AffiliateLink> a bâti sa réputation sur un principe : rendre le cloud simple. Créer un serveur (un "Droplet") prend 30 secondes. Pas de VPC à configurer, pas de groupes de sécurité complexes.

Les prix sont clairs. Un Droplet avec 1 vCPU, 1 Go de RAM et 25 Go SSD coûte 6 dollars par mois. Pas de surprise en fin de mois.

DigitalOcean propose aussi des bases de données managées (PostgreSQL, MySQL, Redis), du Kubernetes managé, du stockage objet (Spaces) et une plateforme d'apps (App Platform) pour déployer sans gérer de serveur.

<Callout type="success" title="Ce qui le rend unique">
Interface simple et bien pensée. Tarification transparente, pas de frais cachés. Documentation parmi les meilleures du secteur (les tutoriels DigitalOcean sont une référence). Kubernetes et bases de données managées à prix raisonnable. 200 dollars de crédit gratuit pendant 60 jours.
</Callout>

<Callout type="warning" title="Le bémol">
Entreprise américaine (Cloud Act). Pas de datacenter en France (le plus proche est à Amsterdam ou Francfort). Moins de services que les hyperscalers. Pas adapté aux très grandes infrastructures. Le réseau est correct mais pas au niveau de Google ou AWS.
</Callout>

## Scaleway : le cloud français nouvelle génération

<AffiliateLink href="https://www.scaleway.com/fr/">Scaleway</AffiliateLink> est une filiale d'Iliad (maison mère de Free). Les datacenters sont à Paris et Amsterdam. L'entreprise se positionne comme une alternative européenne aux géants américains.

L'offre est large : instances cloud, Kubernetes managé (Kapsule), bases de données managées, stockage objet, fonctions serverless, registre de conteneurs. Le tout avec une interface moderne et une API bien faite.

Les prix sont compétitifs. Une instance DEV1-S (2 vCPU, 2 Go RAM, 20 Go SSD) coûte environ 5 euros par mois.

<Callout type="success" title="Ce qui le rend unique">
Entreprise française, datacenters à Paris. Services managés modernes (Kubernetes, serverless, bases de données). API et CLI bien documentées. Engagement écologique (refroidissement adiabatique). Interface de gestion claire et rapide. Free Tier généreux sur plusieurs services.
</Callout>

<Callout type="warning" title="Le bémol">
Moins de régions que les hyperscalers (Paris et Amsterdam principalement). Communauté plus petite, donc moins de tutoriels et de retours d'expérience. Certains services sont encore jeunes. Pas de certification SecNumCloud (contrairement à OVHcloud sur certaines offres).
</Callout>

## Questions fréquentes

### AWS est-il trop cher ?

Ça dépend de l'usage. Pour un VPS basique, oui : AWS est deux à trois fois plus cher que Hetzner ou OVHcloud. Pour des services managés complexes (Lambda, DynamoDB, SageMaker), AWS reste compétitif car les alternatives n'offrent pas toujours d'équivalent.

### Peut-on migrer facilement depuis AWS ?

Si vous utilisez des services standards (VPS, bases de données, stockage objet S3), la migration est faisable. Si vous êtes enfermé dans des services propriétaires AWS (Lambda, Step Functions, DynamoDB), c'est plus compliqué. Utilisez des conteneurs Docker ou Kubernetes pour faciliter la portabilité.

### Quel cloud choisir pour rester en France ?

OVHcloud et Scaleway ont des datacenters à Paris. Azure a des régions en France (Paris, Marseille). Google Cloud n'a pas encore de région française. Hetzner est en Allemagne et Finlande, DigitalOcean au plus proche à Amsterdam.

### Hetzner peut-il remplacer AWS pour une startup ?

Pour beaucoup de startups, oui. Un VPS Hetzner avec Docker et une base de données PostgreSQL couvre 80 % des besoins. Vous économisez des centaines d'euros par mois. Le jour où vous avez besoin de services managés avancés, vous pourrez migrer vers un hyperscaler.

### Google Cloud est-il mieux qu'AWS ?

Ni mieux ni moins bien, mais différent. Google Cloud excelle sur BigQuery, Kubernetes (GKE) et le machine learning (Vertex AI). AWS a plus de services, plus de régions et une communauté plus large. Le choix dépend de votre stack technique.

### La bande passante est-elle vraiment gratuite chez OVHcloud ?

Le trafic sortant est inclus dans le prix des instances OVHcloud, sans limite raisonnable. Chez AWS, chaque Go sortant au-delà du Free Tier est facturé (environ 0,09 $/Go). Sur un site à fort trafic, la différence se chiffre en centaines d'euros par mois.
